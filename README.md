# BuQ — Digital Concierge

Електронне меню та система прийому замовлень для закладів харчування. Працює в режимі реального часу.

## 🔗 Демонстрація
- **Меню (для гостей):** [buq.vua.v.ua](https://buq.vua.v.ua/?table=1)
- **Панель (для персоналу):** [buq.vua.v.ua/admin](https://buq.vua.v.ua/admin)

## 🛠 Технології
- **Frontend:** Nuxt, Vue 3, Tailwind CSS, Pinia
- **Backend / БД:** Supabase (PostgreSQL, Realtime)
- **Деплой:** Cloudflare Pages
